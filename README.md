# Tourism

Семестровая работа - Аэропорт с отелем, где можно:

1. Зарегестрироваться / Войти в свой аккаунт
2. В табло для прилетающих / вылетающиих сортировать рейсы по дате времени, а так же поиск с индексацией.
3. В табло для прилетающих / вылетающиих добавить рейс в профиль
4. На странице отеля выбрать даты, а затем выбрать номер из доступных, и это добавится в профиль
5. Чат с подддержкой
6.В профиле удалять бронь на отель или рейс, изменять доп услуги у брони отеля
